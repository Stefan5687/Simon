# Förderungen Mair – SWOT Analyse

**Erstellt:** 2026-01-28  
**Autor:** Simon

---

## Stärken (Strengths)

| Stärke | Beschreibung |
|--------|--------------|
| **Erfolgsbasiert** | Kein Risiko für Kunden, nur 7-12% bei Erfolg |
| **Schnell** | 2-Minuten-Formular, 3-Tage-Rückmeldung |
| **Kostenlose Erstberatung** | Niedrige Einstiegshürde |
| **Komplettservice** | Recherche, Antrag, Einreichung, Tracking |
| **Persönlich** | Stefan kennt den lokalen Markt (Innichen/Südtirol) |
| **KI-Unterstützung** | Simon kann schnell recherchieren und analysieren |

---

## Schwächen (Weaknesses)

| Schwäche | Beschreibung |
|----------|--------------|
| **Neu am Markt** | Keine Reputation, kein Portfolio |
| **Abhängig von Stefan** | Alles läuft über eine Person |
| **Begrenzte Reichweite** | Aktuell nur regional (Südtirol?) |
| **Rechtliche Komplexität** | Förderungen ändern sich ständig |
| **Cashflow** | Zahlung erst bei Erfolg – lange Wartezeiten |

---

## Chancen (Opportunities)

| Chance | Beschreibung |
|--------|--------------|
| **Wachsender Markt** | Mehr Unternehmen suchen Förderungen |
| **Südtirol-Potenzial** | Regionaler Fokus, wenig lokale Konkurrenz |
| **Erfolgsbasis-Modell** | Differenziert sich von traditionellen Beratern |
| **Automatisierung** | Simon kann Recherche/Antrag unterstützen |
| **Skalierbar** | Online-Modell, keine physische Präsenz nötig |
| **B2B-Netzwerk** | Stefan's Arbeit im Tourismusverein öffnet Türen |

---

## Bedrohungen (Threats)

| Bedrohung | Beschreibung |
|-----------|--------------|
| **Konkurrenz** | Große Beratungsfirmen, Online-Plattformen |
| **Marktveränderung** | Förderprogramme können wegfallen |
| **Rechtliche Risiken** | Falsche Beratung = Haftung |
| **Wirtschaftslage** | Bei Rezession weniger Investitionen |
| **Abhängigkeit** | Förderregeln ändern sich oft |

---

## Fazit

**Kernkompetenz:** Erfolgsbasiertes, schnelles, persönliches Förderungs-Consulting mit KI-Unterstützung.

**Strategie:**
1. Regionalen Fokus ausbauen (Südtirol)
2. Stefan's Netzwerk nutzen
3. Erfolgsgeschichten sammeln
4. Skalierbar machen mit Simon's Hilfe
